# 🖖 oh-StarTrek *(Preview)*

> *"Every task checked off is a warp jump toward a distant star."* ✨

## 🌠 Starfield Canvas

Every editor panel is a viewport into the stars.

- 🪐 **Independent Starfield** — each panel has its own star domain
- 🧲 **Mouse Gravity** — stars near your cursor brighten and pull toward it
- 🌊 **Scroll Parallax** — stars drift slower than your content, as if you're in deep space
- ✦ **Dual-color Stars** — cyan and orange echoing the theme's accent colors

## ⚡ Warp Animation

> *Every task completed is a warp jump toward your destination* 🚀

When you check off a task, the starfield warps for you ✧

- Small task (check a checkbox) → a short, light jump
- Big task (tag as Done) → a full-speed warp

Stars accelerate, streak with light trails, then drift back — as if you've arrived at that destination too ヾ(≧▽≦*)o

## ✨ Morphic Field Integration

When used with the oh-StarTrek-LCARS plugin's Morphic Field feature, stars near the indicator line are pulled by gravity as you drag to resize the editor, streaking with luminous trails; upon release, they drift back and their glow fades.

![Morphic Field + Starfield](images/morphic-field-starfield.gif)

## 🎨 Visual Style

- 🖤 **Borderless Immersion** — no harsh lines, depth defined by light and shadow
- 🌑 **Pure Neutral Grayscale** — zero hue shift, steady as deep space
- 🌈 **Scope-line Breathing** — orange-cyan gradient slowly pulsing like a nebula
- 💫 **Input Glow** — soft radiance on focus
- 🔤 **Font Preview** — dropdown menus show actual font rendering (requires installed fonts)

## ⚙️ Plugin Settings

| Setting | Default | Description |
|---------|---------|-------------|
| Warp Engine 🛸 | On | Starfield master switch. Turn off if laggy, but you'll lose your stars (ಥ_ಥ) |
| Debug Mode | Off | Show FPS, coordinates and other debug info |

## ⚠️ About Preview Version

This is a **Preview version** — light mode has not been optimized yet.

👉 **Dark mode recommended** — the starfield shines brightest in the dark 🌑✨

## 📦 Installation

1. Place the `oh-StarTrek` folder in your Orca Note plugins directory
2. Restart Orca Note and select **oh-StarTrek** in theme settings 🎉
